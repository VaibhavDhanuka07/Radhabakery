# Radha Cooking & Bakery Classes

Premium cooking class website frontend built with React (Vite) and Tailwind CSS.

## Features
- Elegant, minimal, premium design (inspired by MasterClass)
- Responsive layout
- Class cards with hover effects
- Clean navbar and footer
- Protected routes
- Dummy data for classes and users

## Pages
- Home
- About
- Classes (with filter)
- Class Detail
- Login / Signup
- User Dashboard
- Admin Dashboard

## Design
- Brand colors: Soft pink (#E7B6B6), dark text (#2B2B2B), light background (#FFF8F8)
- Serif font for headings
- Soft shadows and spacing
- Simple UI for all age groups

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
